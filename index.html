04.Competitions & research

Team submissions where I led or contributed to the ML/engineering side.

DataQuest 2024 🥈 2nd place (Private LB)

Network Threat Detection via ML Traffic Classification

Team Maya Tilis · AIRNOLOGY 3.0 DataQuest, Fasilkom UI

Multi-class classification of network flow traffic (Benign, Probing, Bruteforce, XMRIGCC CryptoMiner, …) on a highly-imbalanced 416k-row dataset. We engineered ratio & aggregation features over raw flow stats, used error-analysis-driven feature engineering to fix Benign/Probing confusion, and trained CatBoost / LightGBM / XGBoost. Final CatBoost model scored 0.88461 custom metric → 2nd on the private leaderboard.

  • CatBoost
  • Feature Engineering
  • Imbalanced Data
  • EDA

Technical paper

E-Commerce Product Retrieval — Dense Retrieval + Cross-Encoder Reranking + Visual Search

Team Sobat Imut · co-authored research paper

End-to-end search system on the Wayfair WANDS dataset. Compared BM25, bi-encoder dense retrieval (MiniLM-L6), and cross-encoder reranking (ms-marco MiniLM-L12) fine-tuned with contrastive learning. A 41M-param fine-tuned cross-encoder beat the 560M BAAI/bge-reranker-large at NDCG@10 (0.850 vs 0.833). Integrated CLIP for multimodal visual search.

  • Sentence-Transformers
  • Cross-Encoder
  • Contrastive Learning
  • CLIP
  • NDCG

SPARK Telkomsel Idea proposal

TRACE — Traceable Real-time Adaptive Continual Explainable Deepfake Detection

SPARK Telkomsel × ITB × Google · co-authored proposal

System design for real-time deepfake detection across images, recorded video, and live capture. Image CNN encoder + Audio-Visual Feature Fusion (AVFF) pipeline, knowledge-distillation-based continual learning to avoid catastrophic forgetting, LIME for visual explainability, and a Disentanglement Reversing Network (DRN) for tracing the original face. Designed to deploy on Bliv + Vertex AI.

  • PyTorch
  • CNN
  • AVFF
  • Knowledge Distillation
  • LIME
  • Vertex AI
